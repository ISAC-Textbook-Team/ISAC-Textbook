\subsection*{Section 2.2}

\begin{enumerate}[label=\textbf{\arabic*.}, leftmargin=*]
    \item \textbf{Radar Range and Velocity Estimation.}

    In a monostatic radar system, the measured round-trip delay is
    \(\tau=0.8\,\mu\mathrm{s}\). The carrier frequency is
    \(f_c=77\,\mathrm{GHz}\), and the measured Doppler shift is
    \(f_D=3\,\mathrm{kHz}\). Compute the target range and the radial velocity.

    \item \textbf{Range and Velocity Resolution.}

    A radar waveform has bandwidth \(B=500\,\mathrm{MHz}\), carrier frequency
    \(f_c=77\,\mathrm{GHz}\), and coherent observation time
    \(T_{\mathrm{obs}}=20\,\mathrm{ms}\). Compute the approximate range
    resolution and velocity resolution.

    \item \textbf{Angle Estimation with a Uniform Linear Array.}

    A uniform linear array has adjacent antenna spacing \(d=\lambda/2\). If
    the measured phase difference between adjacent antenna elements is
    \(\Delta\phi=\pi/3\), compute the angle of arrival \(\theta\) relative to
    the array broadside direction.

    \item \textbf{Neyman--Pearson Detection Threshold.}

    Consider a scalar detection statistic satisfying
    \(T|H_0\sim\mathcal{N}(0,1)\) and
    \(T|H_1\sim\mathcal{N}(3,1)\). If the false alarm probability constraint is
    \(P_{FA}=0.05\), compute the threshold \(\gamma\) and estimate the
    detection probability \(P_D\).
\end{enumerate}
